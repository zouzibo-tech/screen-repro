# API 速率限制配置

> **说明**：本文件定义当前使用的AI模型的速率限制参数。

> 不同模型有不同的限制，请根据实际情况填写。

## 速率限制参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **最大并发请求数** | 1 | 同时运行的子agent数量 |
| **每分钟最大请求数（RPM）** | {rpm} | API每分钟允许的请求次数 |
| **每分钟最大输入Token数** | {input_tpm} | 每分钟允许的输入token总量 |
| **每分钟最大输出Token数** | {output_tpm} | 每分钟允许的输出token总量 |
| **每次请求最大Token数** | {max_tokens} | 单次请求的token上限 |
| **请求间隔（秒）** | {interval_sec} | 两次请求之间的最小间隔 |

---

## 429错误处理策略

| 策略 | 说明 |
|------|------|
| **自动重试** | 收到429后，等待 `{retry_wait_sec}` 秒后自动重试 |
| **最大重试次数** | {max_retries} 次，超过后标记该文献为"ERROR"并跳过 |
| **退避策略** | 每次重试等待时间翻倍（指数退避）：{retry_wait_sec}秒 → {retry_wait_sec*2}秒 → {retry_wait_sec*4}秒 |
| **批量间隔** | 每处理完1篇后，等待 `{batch_wait_sec}` 秒再处理下一篇 |
